# Jessbella E-commerce Platform

## About The Project
Jessbella is a comprehensive, responsive E-commerce web application developed as a core project for my engineering coursework. The platform is designed to provide a seamless shopping experience, featuring a dynamic inventory management system and real-time order tracking.

The name **"Jessbella"** is a unique brand identity that I conceptualized and developed personally to reflect the aesthetic and functional vision I had for this project.

## Key Features
- **Intuitive UI/UX:** A clean, responsive design optimized for various screen sizes.
- **Dynamic Inventory Management:** Full CRUD (Create, Read, Update, Delete) functionality for product management.
- **Shopping Cart System:** A robust mechanism for users to manage their selections before checkout.
- **Data Persistence:** Integrated MySQL backend for secure handling of products, users, and order history.

## Technologies
- **Frontend:** HTML5, CSS3 (Tailwind CSS), JavaScript.
- **Backend:** PHP.
- **Database:** MySQL.
- **Development Tools:** Figma (for UI/UX prototyping), Cursor & VS Code (for development), XAMPP (local server environment).

## Getting Started
To run this project on your local machine:
1. **Clone the repository:** `git clone ` `https://github.com/noor-abdalfatah/-Jessbella-By-Noor.git`
2. **Database Setup:** Open `phpMyAdmin`, create a database named `ecommerce_db`, and import the file located at `database/database.sql`.
3. **Configuration:** Open `config.php` and update the database credentials (username/password) to match your local server environment.
4. **Launch:** Place the project folder in your `htdocs` directory and open `http://localhost/ecommerce_project/index.php` in your browser.
